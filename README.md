# networking_sfml
Simple sfml server/client example with TCP sockets and listener

Instructions
```bash
    mkdir build
    cd build
    cmake ..
    make
    cd ..
    cp -r Client/Media/ bin/
```


Server dependencies:
- `sfml-system`
- `sfml-network`

Client dependencies:
- `sfml-window`
- `sfml-system`
- `sfml-graphics`
- `sfml-network`
